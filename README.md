# Prime Drop Elite

Plataforma e-commerce de bolsos de lujo. Construida con React 19, TypeScript y Vite.

## Desarrollo local

**Requisitos:** Node.js 18+

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000)

## Build para producción

```bash
npm run build
```

Los archivos estáticos quedan en `/dist` — súbelos a Hostinger vía File Manager o FTP.
